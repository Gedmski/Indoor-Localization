# Archived UJI and Multi-Building Files

This folder contains files moved out of the active Building 10 workflow.

## Why archived

The project was refocused to only support Building 10 (`data/bldg10/final_data.csv`).
Anything related to UJI multi-building experiments, XY regression, or legacy reports
was moved here to keep the active codebase simple.

## Contents

- `archive/legacy_uji/root/`: legacy root docs and old evaluation entrypoint
- `archive/legacy_uji/src/`: legacy model comparison, cross-validation, tuning, and XY utilities
- `archive/legacy_uji/data/`: `TrainingData.csv`, `ValidationData.csv`
- `archive/legacy_uji/models/`: legacy model artifacts
- `archive/legacy_uji/reports/`: legacy logs and reports
- `archive/legacy_uji/notebooks/`: legacy exploratory notebook
